# Planteamiento del Problema

Muchas personas tienen dificultades para organizar sus actividades diarias debido a la falta de herramientas simples y centralizadas para gestionar tareas y compromisos.

Esto provoca pérdida de tiempo, olvidos frecuentes y disminución de productividad.

AgentePETE busca resolver este problema proporcionando una herramienta local que permita registrar, consultar y organizar actividades mediante una interfaz amigable apoyada por inteligencia artificial.
